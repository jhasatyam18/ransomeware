### Prerequisites
DMSystem requires [Node.js](https://nodejs.org/) to run.

### How to run project
Open src/setupProxy.js and set you data-engine server IP.
target: 'https://<PAH>:<PORT>/'

```sh
$ npm install
$ npm run start
```
